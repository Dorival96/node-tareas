## Aplicación de comandos NodeJS

Recuerda instalar los paquetes

```
npm install yargs
```
## Opciones
* Crear Lista
```
node app.js crear
```
* Actualizar estado de la tarea
```
node app.js actualizar
```
* Borrar Tarea
```
node app.js borrar
```
* Listar Tarea
```
node app.js listar
```
Adicional: -d para descripción de la tarea, -c para completado añadir estado de la tarea.
